### ***Description***
The goal of the website is to provide users with knowledge of existing rides and read/write reviews by other amusement park guests.
***

### ***Technologies Used***

* PostgreSQL + Sequelize
* Express.js
* React
* Node.js
* JavaScript



***

### ***Getting Started***
1.  Fork and clone
2.  npm install
3.  npm run dev
